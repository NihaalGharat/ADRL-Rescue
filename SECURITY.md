---

## Reporting a Vulnerability

If you discover a security vulnerability within ADRL-Rescue, please send an email to gharatnihaal@gmail.com. All security vulnerabilities will be promptly addressed.

**Please do not report security vulnerabilities through public GitHub issues.**

---

## Security Guidelines

### For Users

- Keep your Unity installation updated
- Use strong passwords for GitHub accounts
- Enable two-factor authentication on GitHub
- Don't share API keys or credentials

### For Contributors

- Never commit secrets or credentials
- Use environment variables for sensitive data
- Follow secure coding practices
- Review code for vulnerabilities before submitting PRs

---

## Dependency Security

We regularly update dependencies to address known vulnerabilities:

- Unity packages are updated to latest LTS
- Python dependencies are pinned to specific versions
- GitHub Dependabot alerts are monitored
